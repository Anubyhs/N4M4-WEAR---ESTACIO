# Pasta de Imagens

Esta pasta contém as imagens dos destinos do aplicativo.

## Como adicionar imagens:

1. Adicione as imagens nesta pasta com os seguintes nomes:
   - `buenos_aires.jpg` - Imagem de Buenos Aires, Argentina
   - `amazonia.jpg` - Imagem da Amazônia / Aldeia dos Povos Arrarra
   - `joao_pessoa.jpg` - Imagem de João Pessoa, Paraíba
   - `ceu_xamanico.jpg` - Imagem do C.E.U Xamânico em Jacumã

2. Formatos aceitos: `.jpg`, `.jpeg`, `.png`

3. Após adicionar as imagens, execute no terminal:
   ```bash
   flutter pub get
   ```

4. Reinicie o aplicativo para ver as novas imagens.

## Dicas:

- Use imagens de boa qualidade (recomendado: 800x600px ou maior)
- As imagens serão redimensionadas automaticamente para caber nos cards
- Se a imagem não for encontrada, será exibido um ícone de fallback

## Onde encontrar imagens:

- Instagram (salve as imagens que deseja usar)
- Unsplash (https://unsplash.com) - imagens gratuitas
- Pexels (https://pexels.com) - imagens gratuitas
- Suas próprias fotos dos destinos
